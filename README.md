# university-management-system
Rol tabanlı yetkilendirme sistemi aktif akademik bilgi sistemi

Bu proje, akademik süreçleri dijitalleştirmek, öğrenci ve öğretmen etkileşimini güvenli bir platform üzerinden sağlamak amacıyla geliştirilmiş kapsamlı bir Öğrenci Bilgi Sistemi (OBS) simülasyonudur.

Sistem, modern web teknolojileri kullanılarak kullanıcı dostu bir arayüzle tasarlanmış olup, arka planda güçlü bir veri yönetimi ve rol temelli yetkilendirme mimarisi barındırmaktadır.

✨ Temel Özellikler

* Rol Temelli Yetkilendirme: Sisteme giriş yapan kullanıcının yetkisine göre (Öğrenci veya Öğretmen) dinamik olarak değişen paneller.
* Öğrenci Paneli: Öğrencilerin kendi akademik durumlarını görüntüleyebilmesi ve not filtreleme işlemleri.
* Öğretmen Paneli:Akademik personelin kendi yetki alanındaki öğrencilerin not girişlerini, düzenlemelerini ve takibini yapabilmesi.
* Güvenli Oturum Yönetimi:Yetkisiz erişimleri engellemeye yönelik kullanıcı doğrulama (Authentication/Authorization) adımları.

🛠️ Kullanılan Teknolojiler

* Backend: C#, ASP.NET Core MVC
* Frontend: HTML5, CSS3 
* Veritabanı: SQL 
* Mimari: Model-View-Controller (MVC)
